Mini::init

static bool init();

描述：初始化函数。

参数：无

返回：bool

Mini::compress

bool compress(const unsigned char *in, uint32_t insize, unsigned char *out, uint64_t &outsize);

描述：数据压缩。

参数：

const unsigned char *in 需压缩的数据。

uint32_t insize 需压缩的数据长度。

unsigned char *out 压缩后保存的数据指针。

uint64_t &outsize 压缩后保存的数据长度。

返回：bool

Mini::compress_andescape

bool compress_andescape(const unsigned char *in, uint32_t insize, unsigned char *out, uint32_t outsize);

描述：数据压缩和字符替换。

参数：

const unsigned char *in 需压缩的数据。

uint32_t insize 需压缩的数据长度。

unsigned char *out 压缩后保存的数据指针。

uint64_t &outsize 压缩后保存的数据长度。

返回：bool

Mini::decompress

bool decompress(const unsigned char *in, uint32_t insize);

描述：数据解压。

参数：

const unsigned char *in 需解压数据。

uint32_t insize 需解压的数据长度。

返回：bool

Mini::get_decompress_buffer

const unsigned char *get_decompress_buffer();

描述：获得解压后的数据指针。

参数：无

返回：const unsigned char *

uint32_t get_decompress_buffersize() const;

描述：获得解压后的数据长度。

参数：无

返回：uint32_t
