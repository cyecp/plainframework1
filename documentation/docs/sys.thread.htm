总述

该部分主要提供了线程类以及线程锁类。

Thread定义


namespace pf_sys {                                                              
                                                                                
class PF_API Thread {

Thread::start

void start();

描述：线程启动。

参数：无

返回：无

Thread::stop

virtual void stop();

描述：线程停止。

参数：无

返回：无

Thread::exit

void exit(void *retval = NULL);

描述：线程退出。

参数：

void *retval 参数指针。

返回：无

Thread::run

virtual void run();

描述：线程运行。

参数：无

返回：无

Thread::get_id

DWORD|uint64_t get_id();

描述：获得线程ID。

参数：无

返回：DWORD|uint64_t
