总述

性能监视器提供了常用的获取各种性能参数的接口。

定义


namespace pf_performance {                                                      
                                                                                
class PF_API Eyes : public pf_base::Singleton< Eyes> {

::getsingleton_pointer

static Eyes *getsingleton_pointer();

描述：获得监视器单例指针。

参数：无

返回：Eyes *

::getsingleton

static Eyes &getsingleton();

描述：获得监视器单例对象。

参数：无

返回：Eyes &

::printinfo

void printinfo();

描述：性能信息打印函数。

参数：无

返回：无

::activate

void activate();

描述：监视器激活（轮询）函数。

参数：无

返回：无

::tick_forFPS

void tick_forFPS();

描述：为计算帧率的轮询函数（已废弃）。

参数：无

返回：无

::get_onlinecount

uint32_t get_onlinecount() const;

描述：获得在线数量。

参数：无

返回：uint32_t

::get_connectioncount

uint32_t get_connectioncount() const;

描述：获得连接数量。

参数：无

返回：uint32_t

::get_uptraffic

uint64_t get_uptraffic() const;

描述：获得上行（发送）网络数据数量。

参数：无

返回：uint64_t

::get_downtraffic

uint64_t get_downtraffic() const;

描述：获得下行（接收）网络数据数量。

参数：无

返回：uint64_t

::set_sendbytes

void set_sendbytes(uint64_t bytes);

描述：设置发送的网络数据数量（当前秒）。

参数：

uint64_t bytes 数据量。

返回：无

::set_receivebytes

void set_receivebytes(uint64_t bytes);

描述：设置接收的网络数据数量（当前秒）。

参数：

uint64_t bytes 数据量。

返回：无

::set_printinfo_interval

void set_printinfo_interval(uint32_t interval);

描述：设置打印信息的时间间隔（秒）。

参数：

uint32_t interval 间隔时间。

返回：无

::set_onlinecount

void set_onlinecount(uint32_t count);

描述：设置在线数量。

参数：

uint32_t count 在线人数。

返回：无

::set_connectioncount

void set_connectioncount(uint32_t count);

描述：设置连接数量。

参数：

uint32_t count 连接数量。

返回：无

::set_fps

void set_fps(float value);

描述：设置帧率（外部接口可以调用）。

参数：

float value 帧率。

返回：无
